# 🏫 Avicenna School Management System

## MODULE 1: HELPDESK/AQR SYSTEM

**Tools**: Laravel 11, Spatie Permission, Highcharts, DataTables, Bootstrap, Email Notification, Multi-role Authentication, Github.

Helpdesk/AQR (Avicenna Quick Response) is a ticketing system to manage complaints and requests from school members and the general public with automatic routing system to the appropriate PIC.

**Key Features Implemented:**
• Multi-role Authentication System using Spatie Permission for Super Admin, Public Relations, Principal, Head of Administration, Psychologist, Head Psychologist, and Administrative Staff.
• Dynamic PIC Assignment based on unit, education level, and position with automatic routing.
• Ticketing System with auto-routing based on issue category and location.
• Email Notification System for ticket status updates and assignments.
• Analytics Dashboard with interactive charts (ticket status, location distribution, sender statistics, weekly trends).
• Rating & Feedback System for user satisfaction measurement.
• Real-time Tracking System for monitoring ticket progress and resolution time.
• Multi-unit Support (Jagakarsa, Cinere, Pamulang) with multi-level education (KB/TK, SD, SMP, SMA).

**Link**: Github | https://github.com/daffaraka/iav

## MODULE 2: SCHOOL MANAGEMENT SYSTEM

**Tools**: Laravel 11, Spatie Permission, DataTables, Bootstrap, Multi-role Authentication, CMS, Github.

School Management System is a system to manage school data, student achievements, PTN/PTS data, WIG (Wildly Important Goals), and company homepage CMS.

**Key Features Implemented:**
• Master Data Management for School, Unit, Student, and University (PTN/PTS) with CRUD operations.
• Achievement Management System with curation workflow (curated/uncurated status).
• WIG (Wildly Important Goals) Tracking System with lead measures and progress monitoring.
• PTN/PTS Distribution Analytics showing student university placement per academic year.
• Content Management System (CMS) for managing company homepage and public content.
• Multi-unit Architecture supporting three campuses (Jagakarsa, Cinere, Pamulang).
• Multi-level Education System (KB/TK, SD, SMP, SMA) with hierarchical data structure.
• Role-based Access Control (RBAC) for user management and permission handling.
• Employee Management with department, position, and class assignment.

**Link**: Github | https://github.com/daffaraka/iav
