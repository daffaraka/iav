# 🏫 Sistem Manajemen Sekolah Avicenna

Aplikasi berbasis **Laravel** untuk mengelola data sekolah, unit pendidikan, serta pencatatan prestasi siswa. Fitur mencakup : 
1. Data Prestasi
2. Data PTN PTS
3. WIG
4. CMS Beranda Perusahaan 
5. Multi role (soon)
---

## 🚀 Fitur Utama

### 🎓 Master Sekolah
Mengelola data sekolah Avicenna:
- Jagakarsa  
- Cinere  
- Pamulang  

### 🏢 Master Unit
Setiap sekolah memiliki unit pendidikan sesuai jenjang:
- KB/TK  
- SD  
- SMP  
- SMA  

### 🏆 Data Prestasi
Menampilkan daftar prestasi siswa yang terbagi menjadi dua jenis:
- **Prestasi Terkurasi** → sudah diverifikasi oleh admin  
- **Prestasi Tidak Terkurasi** → belum diverifikasi
